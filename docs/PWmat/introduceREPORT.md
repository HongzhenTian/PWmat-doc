<font size=5><center>**REPORT文件介绍**</center></font>

REPORT文件是PWmat主要的计算信息输出文件，它主要包含：①完整的输入参数，②K点信息，③每一步自洽计算信息，④最终计算的能量信息，
⑤本征值，⑥计算总时间，下面将逐一对这些内容进行介绍：

##### ①完整的输入参数
``` 
{
          1           1
    PRECISION  = AUTO       
    IN.PSP1   = O.SG15.PBE.UPF
    JOB       = SCF
    IN.ATOM   = atom.config        
     ...
     ...
     NUM_BLOCKED_PSI=            1
     WF_STORE2DISK  =            0
     NUM_DOS_GRID =         1500	
}
```
这一部分主要是用于进行计算分析的参考，另外把这部分内容复制到etot.input中可以对各种参数进行精细调节来改进计算，关于各参数含义以及设置方法，请参考[视频教程](http://www.pwmat.com/pwmat-resource/course-vedio/20180517.mp4)!
